# 公文

[English](README.md) | 简体中文

公文（Gongwen）是一个 Typst 公文格式化包，可生成符合 GB/T 9704—2012 国家标准的中文公文。

## 功能特点

- **标准规范**：严格遵循 GB/T 9704—2012 格式要求
- **结构完整**：涵盖版头、主体、版记三大部分
- **联合行文**：支持多机关联合行文的规范排版
- **自动分页**：奇数页页码居右、偶数页页码居左
- **灵活配置**：所有参数可选，开箱即用

## 安装使用

### 方法一：通过包管理器

```typst
#import "@local/gongwen:0.1.0": gongwen
```

### 方法二：本地引用

```typst
#import "src/lib.typ": gongwen
```

## 快速上手

```typst
#import "@local/gongwen:0.1.0": gongwen

#show: gongwen.with(
  org: "示例机关",
  title: "关于测试通知的印发",
  to: "各有关单位",
  fawenzihao: "示例发〔2026〕1号",
  fawenjigou: "示例机关",
  chengwenriqi: "2026年3月24日",
)

正文内容将自动应用标准公文格式，包括三号仿宋字体、首行缩进等规范排版。
```

## 参数说明

### 版头部分

| 参数名 | 类型 | 用途 |
|:------|:------|:------|
| `fenhao` | 字符串 | 份号（6 位阿拉伯数字） |
| `miji` | 字符串 | 密级和保密期限，如"绝密★30 年" |
| `jinji` | 字符串 | 紧急程度，如"特急""加急" |
| `org` | 字符串/数组 | 发文机关标志 |
| `fawenzihao` | 字符串 | 发文字号，如"国办发〔2026〕1 号" |
| `qianfaren` | 字符串/数组 | 签发人 |

### 主体部分

| 参数名 | 类型 | 用途 |
|:------|:------|:------|
| `title` | 字符串 | 公文标题 |
| `to` | 字符串/数组 | 主送机关 |
| `body` | 内容 | 正文 |
| `fujian` | 字符串/数组 | 附件说明 |
| `fawenjigou` | 字符串/数组 | 发文机关署名 |
| `chengwenriqi` | 字符串 | 成文日期 |
| `fuzhu` | 字符串 | 附注 |

### 版记部分

| 参数名 | 类型 | 用途 |
|:------|:------|:------|
| `chaosong` | 字符串/数组 | 抄送机关 |
| `yinfajiguan` | 字符串 | 印发机关 |
| `yinfariqi` | 字符串 | 印发日期 |

## 字体要求

本包依赖以下字体，使用前请确保系统已安装：

- **Times New Roman** + **思源宋体（Source Han Serif）**——发文机关标志、标题（红色）
- **FandolFang R**——正文（仿宋）
- **FandolHei**——份号、密级、紧急程度（黑体）
